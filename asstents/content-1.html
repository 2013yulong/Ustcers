开源的HTML5前端框架Amaze UI发布，推进mobile first前端Web方案

2014年6月，云适配发布了开源的HTML5前端框架Amaze UI，目前处于内测期。根据Amaze UI的官网介绍，该框架的特点是mobile first，解决Web应用从PC向多屏适配的问题，兼容国内主流浏览器和平台，并且专注解决中文排版优化问题。

InfoQ: 中文站跟云适配CEO陈本峰进行了交流，了解有关Amaze UI框架的更多细节。

InfoQ：现在前端框架这么多，为什么还要开发Amaze UI？

现在很多网站都拥有PC版，想要拥有移动版页面怎么办？过去大多数企业的做法是重新开发一个基于手机或者其他终端设备的移动客户端。不仅开发周期漫长，还要专门有一个团队来运营。最重要的是，除了像滴滴打车、手机地图、微信等高频工具类的APP客户端应用，大多数低频APP客户端并不被太多用户关注。如今，有一些IT企业、电商企业以及大型媒体，正在采取一种更聪明的方法，采取云适配中的自适应方式，让PC端和移动端统一，拥有相同的用户体验。

以联想为例，我们可以在手机上看到一个简单的页面，页面导航上可以看到一键拨叫、社交分享、位置信息等功能，这些功能都是PC端没有的。点击任意链接，页面反应速度非常快。并且字体大小可根据用户喜好调整。不会看到之前导航栏很乱，字体很小，或者要手动缩放才能看清楚内容的状况。更看不到一些乱七八糟的广告。

据云适配技术发明者，也是美通云动CEO陈本峰介绍：“云适配可以让中小企业最多一天时间内拥有移动版的网页。即使是一些有着定制化需求的企业，一个礼拜时间即可实现移动版应用。而Amaze UI则可以让云适配走向开源，更多用户可以免费使用移动版的前端开发框架。”

Amaze UI由云适配平台推出，是云适配开源跨屏前端开发框架，拥有一套跨屏的前端常用网页组件，任何前端开发者都可以免费使用。开发者只用复制代码便可以将这些跨屏的组件写入自己的网页中，而不需要自己在开发的过程中重写一遍。像小米的MI UI一样，Amaze UI是云适配应用的一部分，是一个针对 HTML 5 开发的轻量级、模块化、移动优先的跨屏前端框架，通过拆分、封装了一些常用的网页组件，比如轮播图框、文字框等等。

“Amaze UI，其实谐音就是让开发者有大把的时间去陪妹子。” 云适配联合创始人高婧笑言。

提起开发者，可谓一把辛酸泪。“工作忙，把妹难”似乎已成为一种职业通病。而Amaze UI可以让更多开发者走出“职业忧伤”，赶超移动互联潮流。从小屏到大屏，企业可以实现所有屏幕的适配。Amaze UI含近 20 个 CSS 组件、10 个 JS 组件，更有 17 款包含近 60 个主题的 Widgets，可快速构建界面出色、体验优秀的跨屏页面。开发人员可以更好地解决中文排版优化问题，还可以针对国内主流浏览器及 App 内置浏览器提供更好的兼容性支持。Amaze UI 非常注重性能，基于轻量的 Zepto.js 开发，并使用 CSS3 来做动画交互，平滑、高效，更适合移动设备，让用户的 Web 应用可以高速载入。简言之，Amaze UI为移动而生，为开发者而建。

拥有最佳移动应用体验，云适配在加速度

为了让云适配技术更快普及，让更多开发者体验到Amaze UI的魅力。美通云动公司加快了市场拓展步伐。美通云动由前微软美国总部IE浏览器核心研发团队成员及移动互联网行业专家在美国西雅图创立，旨在解决网页在不同移动设备屏幕上的适配问题。从2个人到60个人，美通云动的团队大部分为开发人员，这些人员有的是来自海外的归国博士，也有来自本地化的草根型人才。在近2年的发展中，该公司适配的网页达到20多个亿。用户包括微软、联想、百度代理、IDC等大型企业，也包括人民日报、中国青年报、人民网等媒体。

据了解，云适配主要面向B2B企业，盈利模式主要通过200多家渠道在销售。由于美通云动把利润全部分给了渠道商，每年只收几百块钱的平台维护费用，所以很多渠道商纷纷加入。去年年底，该公司已经拿到A轮融资，2014年的目标主要是扩大用户数，Amaze UI虽然免费，但却可以让美通云动公司进一步推动云适配应用的发展，实现用户数的进一步增长。目前，云适配行业还是一个空白，只有一些电商、媒体行业有团队在做移动应用，中国95%的网站都没有实现移动化。一些传统营销型企业根本不会自建移动开发团队，20%下小微企业甚至连网站都没有。未来，这些企业将成为云适配的目标客户群。

尽管，近两年移动搜索、移动广告概念吵得很热，但如果在手机端看到的是PC页面，或在PC页面看到的是手机版网页，这两种体验都给用户带来很差的体验。有数据统计，79%的用户如果在手机上打开PC版的网页会直接离开这个网站，对于一个在移动端做营销的企业来说，不是在增加用户数，而是让用户流失。媒体行业也同样，如果无法监测到移动端的数据，看不到曝光量和点击率，没有办法实现移动端的注册，在移动端做广告是没有任何意义的。云适配可以为用户提供标准的或者定制化的平台。不仅能实现PC和移动应用的同步，网址完全相同，还可以提供一些增值服务。比如：可以定制一些APP特有的应用，包括分享、评论等等。另外，还可以通过图片压缩进行页面提速，可以提升页面访问速度5倍以上。

如果说支付宝的出现引发了银行服务模式的思考，微信的出现让中国移动、中国联通不得不进行业务变革，那么云适配的出现将取代很多APP客户端，让移动应用开发者从容应对移动开发工作。尤其是Amaze UI推出后，可以帮前端开发者节省很多开发的时间。
